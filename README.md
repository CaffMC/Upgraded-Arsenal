UpgradedArsenal is a Minecraft Forge mod that introduces new weapon mechanics, visual effects, sounds, and combat enhancements.

To run this project, you MUST have Minecraft Java with the Forge 1.21 version 51.0.33 (https://files.minecraftforge.net/net/minecraftforge/forge/index_1.21.html)
and also have the .jar file located in build/libs/ in the minecraft mod folder.

Current Weapons:
1. Wither Staff
    -Applies Wither II to enemies
    -Applies Slowness II
    -Has a custom cooldown
    -Plays a custom sound when activated
    -Emits black and purple particles around the target
    -Has durability and balanced stats
    -Fully custom texture made for this project

The mod .jar file can be found in build/libs/
